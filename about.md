# About the Brand
Style Loom combines comfort with luxurious aesthetics. Each collection balances mobility and elegance, with seasonal inspirations like Autumn Fantasy or Timeless Charm.

## Conversation Branching Prompts
- If user says “brand story”: offer to go deeper into Design Process / CEO / Sustainability.
- If user asks one branch, end with: “Want to hear about the others as well?” (short, prose).

## Quick One-Liners (to keep answers tight)
- Design: “From fabric research to multiple fittings, we balance mobility with elegance.”
- CEO: “Olivia M. blends textile tech and global fashion management with artisan collabs.”
- Sustainability: “Organic/recycled materials, limited upcycling, and local community production.”
