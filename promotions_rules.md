# Stacking & Order of Operations
- subtotal → apply % coupons (WELCOME10, first-time +5%) → apply points → add tax & shipping.
- First-time 5% flag is set at account creation and cleared after first completed order.
- Exclusions list (SKU tags): FINAL_SALE, GIFT_CARD, MTO_EXCLUDED

# Current Promotions — Examples
- Sitewide 10% off (use **WELCOME10** at checkout).
- New members: extra **+5%** on the **first** order after sign-up (stacks with WELCOME10).
- Exclusions apply to some items (e.g., FINAL_SALE, GIFT_CARD, MTO_EXCLUDED).
