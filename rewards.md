# Rewards & Membership
- **Bronze**: Earn 1 point per $1 spent
- **Silver**: Earn 1.5 points per $1 spent (≥ $300 / yr)
- **Gold**: Earn 2 points per $1 spent + Free Express Shipping (≥ $800 / yr)
- **VIP**: Earn 2 points per $1 spent + Free Express Shipping + Gifts (≥ $1500 / yr)
- 100 points = $1

## Points Examples
- $120 purchase → Bronze earns 120 pts → $1.20 value toward next order
- Stacking: Promo discounts reduce subtotal before points are awarded

## Upgrade/Earn Cadence
- Tiers are calculated on a rolling 12-month spend window
- Downgrades occur if annual spend falls below tier threshold at review date

## Expiration Examples
- If no earning/redeeming activity for 12 months, all unredeemed points expire
- Reminder pings can be set 30/7/1 days before expiry (opt-in)

## FAQ
- **Q:** Can I combine points with coupon codes?  
  **A:** Yes, unless otherwise noted on a specific promotion  
- **Q:** Do points apply to taxes/shipping?  
  **A:** No, points apply to merchandise subtotal only
